d1 將位置凍結在畫面某處 :
position: fixed;

指定 fixed 的位置 :
right / left : 距離畫面最 右/左 多少 px
top / bottom : 距離畫面最 右/左 多少 px

範例:
<div style="position:fixed; right:0px; top: 200px; width: 200px; height: 200px; background: #cccccc"></div>
d2 滑鼠滑過時的 css : hover
範例 : 滑鼠滑過 div 時, 將 div 的背景顏色改變

<head>
div:hover { background: #cccccc; } </head>
